### GNU Bash Shell
#### 什么Shell
> Shell是系统的用户界面,提供了用户和内核进行交换的接口，它接收用户输入的命令并把它送入内核去执行(命令解释器)
> 可以实现对Linux系统的大部分管理:文件,用户,权限,磁盘,软件,网络...
#### 使用Shell的两种方式
1.输入命令   效率低  适合少量的工作  
2.Shell Script 效率高 适合复杂，重复的工作  
#### Shell语法
> ls -a /home
<h4> 
  命令: 整条Shell命令的主体 </br>
  选项: 会影响、微调命令的行为 //通常以,-,-- </br>
  参数: 命令作用的对象  </br>
</h4>
